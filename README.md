## 🎯 프로젝트 소개

이 프로젝트는 JavaScript를 체계적으로 학습하기 위한 30개의 실습 레슨으로 구성되어 있습니다. 각 레슨은 독립적으로 실행 가능하며, 기본 문법부터 비동기 프로그래밍, Node.js 모듈 시스템, 그리고 실전 프로젝트까지 다양한 주제를 다룹니다.

### 학습 목표

- ✅ JavaScript 기본 문법과 데이터 타입 이해
- ✅ 함수형 프로그래밍 개념 습득
- ✅ 비동기 프로그래밍 (Promise, async/await) 마스터
- ✅ Node.js 핵심 모듈 활용
- ✅ 실전 프로젝트를 통한 실무 능력 향상

## ✨ 주요 기능

- 🎮 **인터랙티브 메뉴**: 터미널 기반의 직관적인 레슨 선택 시스템
- 🎲 **랜덤 이모지**: 매번 다른 이모지로 생동감 있는 사용자 경험 제공
- 📚 **30개의 체계적인 레슨**: 기초부터 고급까지 단계별 학습
- 🔄 **모듈화된 구조**: 각 레슨이 독립적으로 실행 가능
- 🛠️ **자동화 스크립트**: 레슨 파일 자동 생성 도구 포함

## 📁 프로젝트 구조

```
javaScriptEx/
├── README.md                 # 프로젝트 설명 문서
├── main.js                   # 메인 프로그램 (레슨 선택 메뉴)
├── utils.js                  # 유틸리티 함수 (랜덤 이모지 등)
├── make_empty_files.sh       # 레슨 파일 자동 생성 스크립트
└── lessons/                  # 레슨 파일 디렉토리
    ├── lesson01.js          # 레슨 1: 변수 선언과 데이터 타입
    ├── lesson02.js          # 레슨 2: 템플릿 리터럴
    ├── lesson03.js          # 레슨 3: 조건문
    └── ...                  # lesson04.js ~ lesson30.js
```

### 주요 파일 설명

#### `main.js`
- 프로그램의 진입점
- 30개 레슨의 메뉴를 표시하고 사용자 입력을 처리
- 선택된 레슨을 동적으로 로드하여 실행
- 랜덤 이모지를 활용한 사용자 친화적 인터페이스

#### `utils.js`
- 공통 유틸리티 함수 모음
- `ME_EMOJI`: 100개 이상의 다양한 이모지 배열
- `getRandomEmoji()`: 랜덤 이모지 선택 함수

#### `make_empty_files.sh`
- `lessons/` 디렉토리 생성
- `lesson01.js` ~ `lesson30.js` 템플릿 파일 자동 생성
- 기본 구조가 포함된 빈 레슨 파일 생성

#### `lessons/`
- 각 레슨의 실습 코드가 포함된 디렉토리
- 각 파일은 `run()` 함수를 export하여 main.js에서 실행

## 🚀 설치 및 실행

### 필수 요구사항

- Node.js (v12 이상 권장)
- npm (Node.js와 함께 설치됨)

### 설치

```bash
# 프로젝트 클론 또는 다운로드
cd javaScriptEx

# 의존성 설치 (필요한 경우)
npm install
```

### 실행

```bash
# 메인 프로그램 실행
node main.js
```

### 레슨 파일 생성 (선택사항)

```bash
# lessons 디렉토리와 빈 레슨 파일들 생성
bash make_empty_files.sh
```

## 📚 레슨 목록

### 기초 문법 (1-7)
1. **변수 선언과 데이터 타입 출력하기** - var, let, const 및 기본 타입
2. **템플릿 리터럴로 문자열 조합하기** - 백틱과 문자열 보간
3. **조건문으로 점수 평가 프로그램 만들기** - if/else, switch
4. **반복문으로 숫자 합계 계산하기** - for, while, do-while
5. **배열 기본 조작(map, filter) 실습** - 배열 메서드
6. **함수 선언문 vs 화살표 함수 비교** - 함수 정의 방법
7. **객체 생성 및 속성 다루기** - 객체 리터럴과 속성 접근

### 고급 개념 (8-12)
8. **Scope와 Hoisting 동작 실험하기** - 스코프 체인과 호이스팅
9. **Closure 이해를 위한 카운터 함수 만들기** - 클로저 활용
10. **this 바인딩 — call, apply, bind 실습** - this 컨텍스트
11. **Destructuring & Spread 연산자 실습** - 구조 분해와 전개
12. **JSON.parse / JSON.stringify 실습** - JSON 처리

### 비동기 프로그래밍 (13-17)
13. **setTimeout, setInterval 타이머 실습** - 타이머 함수
14. **Promise 기본 — 성공/실패 제어** - Promise 생성과 사용
15. **async/await으로 비동기 흐름 제어하기** - 비동기 함수
16. **fetch로 공공 API 데이터 받아오기** - HTTP 요청
17. **비동기 오류 처리 (try/catch)** - 에러 핸들링

### Node.js 모듈 (18-22)
18. **Node.js 모듈 시스템 이해하기** - require/module.exports
19. **fs 모듈로 파일 읽고 쓰기** - 파일 시스템 조작
20. **path 모듈로 경로 처리 실습** - 경로 관리
21. **http 모듈로 간단한 HTTP 서버 만들기** - 웹 서버 구축
22. **npm 패키지(lodash) 설치 및 사용해 보기** - 외부 패키지 활용

### 실전 프로젝트 (23-26)
23. **Todo List 데이터 관리 로직 (CRUD) 기초** - CRUD 구현
24. **간단한 Logger(로그 기록기) 만들기** - 로깅 시스템
25. **유효성 검사 함수 세트 만들기** - Validation
26. **간단한 사용자 인증 흐름 시뮬레이션** - 인증 로직

### 심화 주제 (27-30)
27. **Prototype vs Class 상속 비교** - 상속 패턴
28. **EventEmitter로 커스텀 이벤트 시스템** - 이벤트 기반 프로그래밍
29. **함수형 프로그래밍 기초(map, reduce)** - 함수형 패러다임
30. **모듈 구조를 갖춘 미니 프로젝트 구성** - 종합 프로젝트

## 🎮 사용 방법

### 1. 프로그램 시작

```bash
node main.js
```

### 2. 레슨 선택

터미널에 표시되는 메뉴에서 1-30 사이의 숫자를 입력하여 원하는 레슨을 선택합니다.

```
====================================
   🎮 JavaScript Lessons (1 ~ 30) 🎮
====================================

01. 변수 선언과 데이터 타입 출력하기
02. 템플릿 리터럴로 문자열 조합하기
...
30. 모듈 구조를 갖춘 미니 프로젝트 구성

  Enter '0' to Exit
------------------------------------

실행할 레슨 번호를 선택하세요 ('0' to Exit):
```

### 3. 레슨 실행

선택한 레슨이 자동으로 실행되며, 결과를 확인할 수 있습니다.

### 4. 메뉴로 돌아가기

레슨 실행 후 엔터를 누르면 메인 메뉴로 돌아갑니다.

### 5. 프로그램 종료

메뉴에서 '0'을 입력하면 프로그램이 종료됩니다.

## 🛠️ 개발 가이드

### 새로운 레슨 추가하기

1. **레슨 파일 생성**
   ```bash
   # lessons 디렉토리에 새 파일 생성
   touch lessons/lesson31.js
   ```

2. **레슨 코드 작성**
   ```javascript
   // lessons/lesson31.js

   function run() {
       console.log('새로운 레슨 시작!');

       // 여기에 레슨 코드 작성

       console.log('레슨 완료!');
   }

   module.exports = { run };
   ```

3. **main.js에 레슨 등록**
   ```javascript
   const lessons = [
       // ... 기존 레슨들
       { id: 31, title: '새로운 레슨 제목', file: './lessons/lesson31' },
   ];
   ```

### 유틸리티 함수 추가하기

`utils.js` 파일에 공통으로 사용할 함수를 추가하고 export합니다:

```javascript
// utils.js
function newUtilityFunction() {
    // 함수 구현
}

module.exports = {
    ME_EMOJI,
    getRandomEmoji,
    newUtilityFunction  // 새 함수 추가
};
```

### 코드 스타일 가이드

- **들여쓰기**: 4 spaces
- **따옴표**: 작은따옴표(`'`) 사용
- **세미콜론**: 문장 끝에 세미콜론 사용
- **함수 네이밍**: camelCase 사용
- **주석**: 복잡한 로직에는 설명 주석 추가
